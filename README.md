# LLM From Scratch

## Project Objective
To understand and implement the complete pipeline of a Large Language Model (LLM) from scratch.

## Topics Covered

- NLP Evolution
- Rule-Based Systems
- Statistical NLP
- Word Embeddings
- RNNs
- LSTMs / GRUs
- Attention Mechanism
- Transformers
- Tokenization
- Embeddings
- Training
- Inference
- Text Generation
