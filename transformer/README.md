# Transformer

This folder contains the implementation of the core building blocks of the Transformer architecture based on the paper **Attention Is All You Need (Vaswani et al., 2017)**.

## Implementations

### feed_forward.py
A basic NumPy implementation of the Position-wise Feed Forward Network (FFN).

Features:
- Linear Transformation
- Weight Matrix
- ReLU Activation
